## 방법 1: Firestore에서 직접 수정 (가장 간단)

### 1단계: Firebase Console 접속

1. **Firebase Console 접속**
   - https://console.firebase.google.com/ 접속
   - 프로젝트 선택 (`groupbuying-5d9c5`)

2. **Firestore Database 열기**
   - 왼쪽 메뉴에서 "Firestore Database" 클릭
   - "데이터" 탭 클릭

### 2단계: 사용자 문서 찾기

1. **users 컬렉션 찾기**
   - `users` 컬렉션 클릭
   - 문서 목록에서 `kakao_4706145116` 문서 찾기
   - (최근에 생성된 문서이므로 맨 위에 있을 수 있습니다)

2. **문서 열기**
   - `kakao_4706145116` 문서 클릭

### 3단계: role 필드 수정

1. **role 필드 찾기**
   - 문서에서 `role` 필드 찾기
   - 현재 값: `user`

2. **role 값 변경**
   - `role` 필드 클릭
   - 값 변경: `user` → `admin`
   - 저장 (또는 Enter 키)

### 4단계: 확인

1. **브라우저 새로고침**
   - http://localhost:3000 접속
   - 페이지 새로고침 (F5 또는 Ctrl+R)

2. **관리자 페이지 접속**
   - http://localhost:3000/admin 접속
   - 관리자 대시보드가 보이면 성공!

## 방법 2: 관리자 페이지에서 변경 (관리자 권한 필요)

관리자 권한이 있으면 다른 사용자의 역할을 변경할 수 있습니다:

1. **관리자 페이지 접속**
   - http://localhost:3000/admin 접속

2. **사용자 관리 탭**
   - "사용자 관리" 탭 클릭

3. **역할 변경**
   - 사용자 목록에서 해당 사용자 찾기

   - 역할이 자동으로 변경됩니다
