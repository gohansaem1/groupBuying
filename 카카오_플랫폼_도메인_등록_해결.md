# 카카오 "등록되지 않은 플랫폼" 에러 해결 가이드

## 🔴 현재 에러
```
등록되지 않은 플랫폼에서 액세스 토큰을 요청했습니다.
사용한 JavaScript SDK 도메인: https://group-buying-nine.vercel.app
```

이 에러는 카카오 개발자 콘솔에서 **사이트 도메인**이 제대로 등록되지 않았을 때 발생합니다.

---

## ✅ 해결 방법

### 1단계: 카카오 개발자 콘솔 접속

1. [카카오 개발자 콘솔](https://developers.kakao.com/) 접속
2. 로그인 후 **내 애플리케이션** 선택
3. 해당 앱 선택

---

### 2단계: 플랫폼 설정 확인 및 수정

1. **앱 설정** → **플랫폼** 메뉴로 이동

2. **Web 플랫폼** 섹션 확인
   - 이미 등록되어 있다면 **수정** 버튼 클릭
   - 없다면 **Web 플랫폼 등록** 클릭

3. **사이트 도메인** 입력 방법:

   #### 방법 1: 프로토콜 없이 도메인만 입력 (권장)
   ```
   group-buying-nine.vercel.app
   ```
   - `https://` 제외
   - `/` 제외
   - 도메인만 입력

   #### 방법 2: 전체 URL 입력 (방법 1이 안 되면 시도)
   ```
   https://group-buying-nine.vercel.app
   ```
   - `https://` 포함
   - 전체 URL 입력

4. **저장** 버튼 클릭

---

### 3단계: 여러 형식으로 시도 (필요시)

카카오 개발자 콘솔의 버전에 따라 형식이 다를 수 있으므로, 다음을 모두 시도해보세요:

#### 시도 1: 도메인만 (가장 일반적)
```
group-buying-nine.vercel.app
```

#### 시도 2: 프로토콜 포함
```
https://group-buying-nine.vercel.app
```

#### 시도 3: 포트 번호 없이 (이미 없지만 확인)
```
group-buying-nine.vercel.app
```

---

### 4단계: 기존 등록 삭제 후 재등록 (필요시)

1. **앱 설정** → **플랫폼** 메뉴
2. 기존 Web 플랫폼이 있다면 **삭제** 버튼 클릭
3. **Web 플랫폼 등록** 클릭
4. **사이트 도메인**에 `group-buying-nine.vercel.app` 입력
5. **저장** 클릭

---

### 5단계: Redirect URI도 확인

1. **제품 설정** → **카카오 로그인** 메뉴로 이동
2. **Redirect URI** 섹션 확인
3. 다음 URI가 등록되어 있는지 확인:
   ```
   https://group-buying-nine.vercel.app/login
   ```
4. 없다면 **URI 추가** 클릭하여 추가
5. **저장** 클릭

---

## 🔄 설정 후 테스트

1. **브라우저 캐시 삭제** (중요!)
   - Ctrl + Shift + R (강력 새로고침)
   - 또는 브라우저 개발자 도구 → Application → Clear storage

2. 배포된 사이트 접속: `https://group-buying-nine.vercel.app/login`

3. 카카오 로그인 버튼 클릭

4. 에러가 사라졌는지 확인

---

## 🐛 여전히 에러가 발생하는 경우

### 1. 브라우저 캐시 문제
- **강력 새로고침**: Ctrl + Shift + R
- **시크릿 모드**에서 테스트
- **브라우저 캐시 완전 삭제**

### 2. 카카오 개발자 콘솔 설정 확인
- 플랫폼 설정에서 도메인이 정확히 입력되었는지 확인
- 저장 버튼을 눌렀는지 확인
- 페이지를 새로고침하여 설정이 저장되었는지 확인

### 3. 도메인 형식 확인
- 공백이 없는지 확인
- 오타가 없는지 확인
- `http://`가 아닌 `https://`인지 확인 (Vercel은 자동 HTTPS)

### 4. 여러 도메인 등록 시도
- 로컬 개발용: `http://localhost:3000` (별도로 유지)
- 배포용: `group-buying-nine.vercel.app` (새로 추가)

---

## 📝 참고사항

1. **변경사항 반영 시간**: 카카오 개발자 콘솔 설정 변경 후 즉시 반영됩니다.

2. **로컬 개발 환경**: 
   - 로컬 개발 시에는 `http://localhost:3000`을 별도로 등록해야 합니다.
   - 배포 환경과 로컬 환경은 각각 다른 사이트 도메인을 사용합니다.

3. **Vercel 프리뷰 배포**: 
   - Vercel은 각 PR마다 프리뷰 URL을 생성합니다 (예: `group-buying-nine-git-branch.vercel.app`)
   - 프리뷰 URL도 사용한다면 별도로 등록해야 합니다.
