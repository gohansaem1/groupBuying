# 포트 3000 등록 가이드 (긴급!)

현재 오류: "등록되지 않은 플랫폼에서 액세스 토큰을 요청했습니다. 사용한 JavaScript SDK 도메인: http://localhost:3000"

## 문제

카카오 개발자 콘솔에 **포트 3000**이 등록되지 않았습니다!

## 즉시 해결 방법

### 1단계: 카카오 개발자 콘솔 접속

1. https://developers.kakao.com/ 접속
2. 내 애플리케이션 클릭
3. 앱 선택

### 2단계: 플랫폼에 포트 3000 추가

1. **앱 설정** → **플랫폼** 탭 클릭

2. **웹 플랫폼 추가**
   - "웹 플랫폼 등록" 클릭 (또는 기존 웹 플랫폼 수정)
   - 사이트 도메인에 **포트 3000 추가**:
     ```
     http://localhost:3000
     ```
   - **저장** 클릭

   **중요:**
   - 포트 3000과 3001을 **둘 다** 등록해야 합니다
   - 각각 따로 등록하거나, 여러 개를 한 번에 등록할 수 있습니다

### 3단계: Redirect URI에도 포트 3000 추가

1. **제품 설정** → **카카오 로그인** 클릭

2. **Redirect URI 등록**
   - "Redirect URI 등록" 클릭
   - 다음 URI들이 모두 등록되어 있는지 확인:
     ```
     http://localhost:3000
     http://localhost:3001
     ```
   - 없으면 추가하고 **저장** 클릭

### 4단계: 저장 및 대기

1. **모든 설정 저장 확인**
2. **1-2분 대기** (카카오 서버에 반영 시간)

### 5단계: 테스트

1. **브라우저 캐시 삭제**
   - Ctrl + Shift + R (강력 새로고침)

2. **다시 로그인 시도**
   - http://localhost:3000 접속
   - "카카오 로그인" 버튼 클릭

## 참고

- 개발 중 포트가 변경될 수 있으므로 **여러 포트를 미리 등록**해두는 것이 좋습니다
- 포트 3000, 3001, 3002 등을 모두 등록해두면 편리합니다
- 프로덕션 배포 시 실제 도메인도 추가로 등록해야 합니다
