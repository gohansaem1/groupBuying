# 카카오 개발자 콘솔 메뉴 위치 가이드

## 📍 플랫폼 설정 메뉴 찾기

### 전체 경로
1. [카카오 개발자 콘솔](https://developers.kakao.com/) 접속
2. 로그인
3. **내 애플리케이션** 클릭 (왼쪽 상단 또는 메인 화면)
4. 앱 선택 (앱 목록에서 클릭)

---

## 🎯 앱 선택 후 화면 구조

### 왼쪽 사이드바 메뉴 (주요 메뉴)

카카오 개발자 콘솔의 왼쪽에 다음과 같은 메뉴들이 있습니다:

```
📱 내 애플리케이션
  └─ [앱 이름] 선택
      ├─ 📊 대시보드
      ├─ ⚙️ 앱 설정
      │   ├─ 일반
      │   ├─ 플랫폼  ← 여기!
      │   └─ 앱 키
      ├─ 🛠️ 제품 설정
      │   ├─ 카카오 로그인
      │   ├─ 카카오톡 메시지
      │   └─ ...
      └─ ...
```

---

## 🔍 단계별 찾기 방법

### 방법 1: 왼쪽 사이드바에서 찾기

1. **앱 선택 후** 왼쪽 사이드바를 확인
2. **"앱 설정"** 메뉴 클릭 (톱니바퀴 아이콘 ⚙️)
3. **"플랫폼"** 탭 클릭
4. **"Web 플랫폼 등록"** 또는 **"웹 플랫폼 등록"** 버튼 클릭

---

### 방법 2: 상단 탭에서 찾기

일부 카카오 개발자 콘솔 버전에서는:

1. 앱 선택 후 상단에 탭 메뉴가 있음:
   ```
   [대시보드] [앱 설정] [제품 설정] [통계] ...
   ```
2. **"앱 설정"** 탭 클릭
3. 하위 메뉴에서 **"플랫폼"** 클릭
4. **"Web 플랫폼 등록"** 버튼 클릭

---

### 방법 3: 직접 URL로 접근

앱 ID를 알고 있다면:

1. 브라우저 주소창에 다음 형식으로 입력:
   ```
   https://developers.kakao.com/console/app/{앱키}/platform
   ```
   예: `https://developers.kakao.com/console/app/123456/platform`

2. 또는 앱 선택 후 URL 끝에 `/platform` 추가

---

## 📸 화면에서 찾는 방법

### 플랫폼 설정 화면의 특징

플랫폼 설정 화면에는 다음과 같은 내용이 표시됩니다:

- **플랫폼 목록**:
  - Web 플랫폼
  - iOS 플랫폼
  - Android 플랫폼
  - 등등...

- **Web 플랫폼 섹션**:
  - "웹 플랫폼 등록" 또는 "Web 플랫폼 등록" 버튼
  - 또는 "사이트 도메인" 입력 필드

---

## 🔄 카카오 개발자 콘솔 UI 변경사항

카카오 개발자 콘솔의 UI가 업데이트되면서 메뉴 위치가 변경될 수 있습니다:

### 구버전 UI
- 왼쪽 사이드바에 "앱 설정" → "플랫폼"

### 신버전 UI
- 상단 탭: "앱 설정" → 하위 메뉴 "플랫폼"
- 또는 왼쪽 사이드바: "앱 설정" → "플랫폼"

---

## 💡 찾기 팁

### 1. 검색 기능 사용
- 카카오 개발자 콘솔 내에서 "플랫폼" 또는 "platform" 검색
- 또는 "사이트 도메인" 검색

### 2. 앱 설정 메뉴 확인
- "앱 설정" 메뉴를 클릭하면 하위 메뉴가 나타남:
  - 일반
  - **플랫폼** ← 여기!
  - 앱 키

### 3. 브라우저 개발자 도구 사용
- F12 키로 개발자 도구 열기
- Elements 탭에서 "플랫폼" 또는 "platform" 검색

---

## 🎯 정확한 위치 (최신 UI 기준)

### 경로 1: 왼쪽 사이드바
```
내 애플리케이션
  └─ [앱 이름]
      └─ 앱 설정 (⚙️)
          └─ 플랫폼
              └─ Web 플랫폼 등록
```

### 경로 2: 상단 탭
```
[앱 이름] 페이지
  └─ 상단 탭: [대시보드] [앱 설정] [제품 설정] ...
      └─ 앱 설정 탭 클릭
          └─ 플랫폼 메뉴 클릭
              └─ Web 플랫폼 등록
```

---

## ✅ 확인 방법

플랫폼 설정 화면에 도달했다면 다음을 확인할 수 있습니다:

- ✅ "Web 플랫폼" 섹션이 보임
- ✅ "사이트 도메인" 입력 필드가 있음
- ✅ "웹 플랫폼 등록" 또는 "Web 플랫폼 등록" 버튼이 있음
- ✅ 기존에 등록된 도메인 목록이 보일 수 있음
